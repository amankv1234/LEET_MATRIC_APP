# LeetMatric

🚀 A web app that shows **LeetCode user statistics** by entering a valid username.  
It displays:
- ✅ Total problems solved
- 🟢 Easy solved
- 🟡 Medium solved
- 🔴 Hard solved

## 🔧 Tech Stack
- HTML, CSS, JavaScript
- LeetCode API
